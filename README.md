# "Settling the Size Matter" paper

This repository contains the Python code to replicate the results of the "Settling the Size Matter" paper by Matthias Hanauer and David Blitz. Run each exhibit notebook independently to get the results.

## Blog Post

https://sgokhales.github.io/posts/settling-the-size-matter/

## Original Paper

Blitz, David and Hanauer, Matthias Xaver, Settling the Size Matter (September 4, 2020). Available at SSRN: https://ssrn.com/abstract=3686583 or http://dx.doi.org/10.2139/ssrn.3686583
